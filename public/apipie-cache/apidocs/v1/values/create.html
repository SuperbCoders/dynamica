POST /api/v1/projects/:project_id/values
Upload values

It uploads values to the project.

It accepts item description and array of values. If there is no item with the specified SKU it will be created.

Successfull request responds with 201 status code and returns array of uploaded values. Otherwise see error codes list.

Errors

422 Unprocessable entity
403 Access denied
500 Server error

Examples

// Successful request

// POST /api/v1/projects/my-shop/values

{
  {
  "item":{
    "sku":"shampoo",
    "name":"Shampoo for any type of hair"
  },
  "values":[
    {
      "value":1,
      "timestamp":"2015-01-15 10:20:11"
    },
    {
      "value":2,
      "timestamp":"2015-01-16 09:00:17"
    }
  ]
}

// Respond

[
  {
    "value":{
      "value":1.0,
      "timestamp":"2015-01-15T10:20:11.000Z"
    }
  },
  {
    "value":{
      "value":2.0,
      "timestamp":"2015-01-16T09:00:17Z"
    }
  }
]

// Request with invalid data

// POST /api/v1/projects/my-shop/values

{
  "item":{
    "sku":"shampoo",
    "name":"Shampoo for any type of hair"
  },
  "values":[
    {
      "value":1
    }
  ]
}

// Respond

[
  {
    "timestamp":[
      "can't be blank"
    ]
  }
]
