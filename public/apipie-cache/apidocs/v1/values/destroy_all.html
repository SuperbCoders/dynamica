DELETE /api/v1/projects/:project_id/items/:item_id/values
Destroy all the values of the selected item

Successfull request destroys all the value of the specified item and respond with 204 status code

Errors

403 Access denied
500 Server error

Examples

// DELETE /v1/projects/my-shop/items/shampoo/values
// Responds with empty body and 204 status code
